What You Need To Know

A Buyer in OrderCloud is an organization or set of related users that make orders to a distributor (wholesaler, manufacturer, etc.) through an OrderCloud application, having a similar level of access and using a similar set of purchasing rules. A Buyer is one of the foundations of an OrderCloud application, because it often provides the context in which some other object is being defined (such as a user or category).

Creating a Buyer is one of the most straightforward API Requests you will make in OrderCloud. In fact, you are only required to provide a name of your new buyer when creating one. If you choose to do this, your new buyer's ID will be auto-generated, and Active will be set to false. Creating a buyer uses the Buyers service provided from the SDK. The Buyers service, in turn, provides all the standard CRUD operations for dealing with Buyers.
