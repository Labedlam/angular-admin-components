What You Need To Know

From this point on, it is important that you remember that every API call you make requires that an authentication token be set. As you know from the previous class on authentication, this token is stored in your browser cookies.

Creating a Buyer is very easy. In fact, you are only required to provide a name of your new buyer when creating one. If you choose to do this, your new buyer's buyerID will be auto-generated, and 'Active' will be set to false. Creating a buyer uses the 'Buyers' service provided from the SDK. The 'Buyers' service is almost exclusively a set of the standard CRUD operations for buyers.
